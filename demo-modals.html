Ejemplos de Código

Confirmación:

const confirmed = await DESSModal.confirm('¿Seguro que quieres continuar?', 'Confirmar Acción');
if (confirmed) {
    console.log('Usuario confirmó');
}

Alerta de éxito:

await DESSModal.success('Operación completada exitosamente');

Toast de información:

DESSModal.toast('Información guardada', 'info', 3000);
